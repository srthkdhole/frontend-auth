# 🎨 Authentication App (Frontend)

## 🚀 Overview

This is the frontend of a full-stack authentication system.

Users can log in, view their profile, and log out securely.

## 🛠 Tech Stack

* React
* JavaScript
* CSS

## 🔥 Features

* Login UI
* API Integration with backend
* Token storage using localStorage
* Fetch user profile
* Logout functionality
